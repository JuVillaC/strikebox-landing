# 🥊 Strikebox Legend Factory Concepción - Landing Page

Esta es una Landing Page de alto impacto diseñada y desarrollada para **Strikebox Concepción**, un dojo especializado en Brazilian Jiu-Jitsu, Boxeo, MMA y Kickboxing. El objetivo principal del proyecto es la captación de nuevos alumnos mediante una experiencia de usuario (UX) optimizada y una identidad visual ruda y profesional.

## 🚀 Características Principales

- **Horarios Inteligentes (Smart Schedule):** El sistema detecta automáticamente el día actual de la semana y muestra solo las clases correspondientes, facilitando la consulta rápida desde dispositivos móviles.
- **Filtros Cruzados:** Permite filtrar las clases por disciplina (BJJ, MMA, Kickboxing, Boxeo, Infancias) sin recargar la página.
- **Conversión Directa a WhatsApp:** Todos los botones de llamado a la acción (CTA) están integrados con la API de WhatsApp, incluyendo mensajes predefinidos según el plan de interés.
- **Branding Personalizado:** Identidad visual basada en el color "Naranja Strikebox" (#FF4D00) y estética "Dark Mode" para resaltar la potencia de la marca.
- **Diseño 100% Responsivo:** Optimizado para una visualización perfecta en smartphones, tablets y desktop.
- **Social Proof:** Integración directa con el feed de Instagram para mostrar la comunidad activa del dojo.

## 🛠️ Stack Tecnológico

Este proyecto fue construido bajo la filosofía **Vanilla Web Development**, priorizando la velocidad de carga y la limpieza del código sin dependencias externas pesadas:

- **HTML5:** Estructura semántica avanzada.
- **CSS3:** Diseño responsivo con Flexbox, CSS Grid y variables nativas.
- **Vanilla JavaScript:** Lógica de filtrado de horarios y manipulación del DOM en tiempo real.
- **Google Fonts:** Tipografía "Oswald" para títulos agresivos y "Roboto" para lectura fluida.
